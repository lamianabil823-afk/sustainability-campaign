# حملة الاستدامة الطبيعية

موقع ويب تفاعلي لحملة الاستدامة البيئية، مصمم لتوعية المجتمع بأهمية الحفاظ على البيئة وتبني ممارسات مستدامة.

## المميزات

- تصميم مستجيب يعمل على جميع الأجهزة
- ألوان مستوحاة من الطبيعة (أخضر وبنى)
- تخطيط قصة التمرير
- أقسام متكاملة: نظرة عامة، مدونة، توصيات، اتصل بنا
- نماذج اتصال تفاعلية

## التقنيات المستخدمة

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript (ES6+)
- خط Tajawal العربي

## كيفية التشغيل

1. قم بفك ضغط الملفات
2. افتح ملف `index.html` في متصفحك
3. أو انشئ موقع على GitHub Pages لاستضافته
